# 数组 `array`: `[类型; 长度]`
**使用方式**
- **创建**  
  - **直接赋值**  
    `括号 []` 包裹, 每个元素之间用 `逗号 ,` 分隔  
    (和 tuple 不一样, 若只有一个元素, 元素末尾不必加上 逗号)
  - **批量赋值**  
    `[值; 长度]`
- **访问**  
(坐标 和 Range 必须能在编译时得到)
  - 单个元素 `arr[坐标]` 
  - 引用切片 `&arr[Range<i32>]` (`Range` 参见下文)
  - 借用切片 `&mut arr[Range<i32>]`  
  

**类型注解**  
和 批量赋值的创建方式 类似, 但把每个元素换成对应的类型, 
即 `[类型; 长度]`

**常用方法和属性**
- 方法
  - map 迭代: `.map(f: FnMut)`

> **注意**
> 1. 每个位置的 **类型不能改变**
> 2. 元组的 **长度也不能改变**
> 3. 一个元组中只能有 **1 种类型**
> 4. **不能嵌套**
