## `never`: `!`
**类型**  
在标准库中称为 `never` 类型, 即没有值的类型,  
一般用于 一定会 panic 的函数的返回值,  
即 "没有返回值的函数" 的返回值
  
**示例**
```rust, editable
fn main(){
    fn a(){
        panic!();
    }
    fn b() -> !{
        panic!();
    }

    // 无法输出 ! 类型
    // 但上面的内容通过了编译, 即返回值 ! 通过了类型检查
}
```
```rust, editable
fn main(){
    fn a() -> !{
        1 + 2;
    }

    // 函数内部没有 panic, 故不适用于 ! 返回值
    // 类型匹配错误, 编译报错
}
```
